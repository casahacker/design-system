usar em um projeto

Importe o CSS base, use os tokens via CSS custom properties, e componha com as classes utilitárias.

// 1. importe o stylesheet base @import "@casahacker/ds/styles.css"; // 2. use os tokens em seus próprios componentes .my-button { background: var(--ch-code); color: var(--text-on-color); font: var(--label-02); padding: var(--spacing-04) var(--spacing-05); }
