hover & tooltip

01

Tooltip aparece on-hover (desktop) ou tap (mobile). Conteúdo: label + valor exato + comparação opcional. Passa o mouse no chart abaixo.

legend toggle

02

Clique na legenda esconde/mostra a série. Visual: opacidade 0.3 quando "off". Implementação JS opcional — em SVG nativo, usar display: none ou opacity.

drill-down

03

Click numa barra/fatia abre detalhamento. Usar quando há hierarquia (mês → semana → dia). Sinalize com cursor pointer + tooltip "clica pra detalhar".

brush selection (range)

04

Em line charts longos, brush no eixo X permite zoom em sub-range. Mostra o range selecionado destacado, resto fica fade. Equivalente teclado: input de date range acima.

time range selector

05

Quando dashboard tem múltiplos charts com mesmo eixo temporal, controle único no topo.

alternativas via teclado

08
  • Hover → equivalente: focus via Tab no ponto/barra
  • Drill-down → Enter no elemento focado
  • Brush → input de data range (input type='date')
  • Legend toggle → checkbox associado
  • Time range → button group nativo
