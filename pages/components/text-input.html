quando usar

05

use

  • Sempre com <label> associado
  • Placeholder como exemplo, não como label
  • Helper text dá contexto, erro corrige

não use

  • Placeholder substituindo label
  • Validar a cada keystroke — espera blur
  • Input sem label — quebra a11y

acessibilidade

06 · wcag aa
  • Label associado por for/id
  • Helper antes do erro (não substituído)
  • Erro associado por aria-describedby
  • Required com required attribute
  • Foco visível 2px
