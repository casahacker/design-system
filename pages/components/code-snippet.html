comportamentos

03
  • Botão 'copy' aparece automaticamente (injetado pelo shell.js)
  • Click copia o texto puro (sem tags HTML)
  • Feedback visual 'copied!' por 1.5s
  • Funciona via Clipboard API (HTTPS ou localhost)

quando usar

05

use

  • Trechos de código de exemplo
  • Comandos de terminal
  • URLs ou paths
  • Configurações em formato chave-valor

não use

  • Logs longos — usa um viewer dedicado
  • Conteúdo crítico que precisa ser editável
  • Substituir explicação textual
