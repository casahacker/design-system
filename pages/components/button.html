variantes

02

Primary

Ação mais importante de uma página. Use no máximo uma por contexto.

Secondary

Ação alternativa de mesmo peso. Usada junto ao primary.

Tertiary

Ação de baixa ênfase. Outlined, sem fundo.

Ghost

Mínimo destaque visual. Sem fundo nem borda.

Danger

Ação destrutiva — deletar, remover, cancelar permanente.

quando usar

07

use

  • Um primary por contexto
  • Verbo no infinitivo: salvar, enviar, publicar
  • Danger só pra ação destrutiva irreversível
  • Ghost pra ações secundárias dentro de cards

não use

  • "clique aqui" como label — usa o verbo da ação
  • Múltiplos primary numa mesma seção
  • Danger pra cancelar formulário (use ghost)
  • Botão pra navegação — usa link standalone

acessibilidade

08 · wcag aa
  • Elemento <button> nativo, não div
  • type="button" explícito (default é submit)
  • Foco visível com outline 2px
  • Disabled: aria-disabled true OU disabled attribute
  • Botão só com ícone tem aria-label

código

09 · html
<!-- variantes --> <button type="button" class="btn btn--primary">salvar</button> <button type="button" class="btn btn--secondary">cancelar</button> <button type="button" class="btn btn--danger">deletar</button> <!-- tamanho --> <button class="btn btn--primary btn--sm">small</button> <!-- icon-only --> <button class="btn btn--ghost btn--icon" aria-label="fechar">×</button>
