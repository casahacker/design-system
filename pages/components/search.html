05
  • X aparece quando há valor (futuro — depende do browser)

quando usar

06

use

  • Busca global no app/site
  • Filtragem de lista longa
  • Ações que envolvem keyword/query

não use

  • Filtros estruturados — usa dropdowns/checkboxes
  • Input genérico — usa text-input

acessibilidade

07 · wcag aa
  • type="search" semântico
  • aria-label='buscar' ou label associado
  • Foco visível
  • role="search" no container quando relevante
