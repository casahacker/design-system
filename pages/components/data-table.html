quando usar

05

use

  • Dados tabulares com colunas relacionadas
  • Comparação entre linhas
  • Listagens com 5+ campos

não use

  • Layout — usa CSS grid/flex
  • Listas simples (1-2 colunas) — usa <ul>
  • Dashboards com cards — usa tile grid

acessibilidade

06 · wcag aa
  • <table> semântico, não divs
  • <th scope="col"> em headers
  • <caption> descrevendo a tabela (visível ou .sr-only)
  • Responsivo: scroll horizontal em mobile ou transformação em cards
