quando usar

06

use

  • Valores contínuos (brilho, volume)
  • Quando precisão exata não é crítica
  • Quando visualização do range ajuda (zoom, escala)

não use

  • Valores discretos pequenos (1-5) — usa radio
  • Quando precisão é crítica — usa number input
  • Múltiplos sliders empilhados sem labels

acessibilidade

07 · wcag aa
  • role="slider" implícito via input[type="range"]
  • aria-valuemin/max/now implícitos
  • Funciona 100% via teclado (setas + Home/End/PageUp/Down)
  • Foco visível com box-shadow no thumb
  • Sempre acompanhar de label associado

código

08 · html
<div class="slider-row"> <div class="slider"> <input type="range" min="0" max="100" value="60"> </div> <div class="slider-value">60</div> </div>
