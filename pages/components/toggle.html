quando usar

06

use

  • Preferências binárias com efeito imediato (notificações on/off)
  • Settings de configuração
  • Opção isolada (1 toggle por linha)

não use

  • Form com múltiplos campos esperando submit — usa checkbox
  • Escolha entre 2+ opções — usa radio
  • Ação destrutiva — usa button confirmação

acessibilidade

07 · wcag aa
  • role="switch" implícito via input[type="checkbox"]
  • aria-checked reflete estado
  • Funciona via Tab + Space
  • Foco visível com outline 2px
  • Label clicável (toda a área .toggle)

código

08 · html
