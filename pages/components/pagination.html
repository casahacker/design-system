quando usar

05

use

  • Listas com 50+ itens
  • Tabelas paginadas no servidor
  • Quando paginação é mais clara que scroll infinito

não use

  • Listas com <30 itens — mostra tudo
  • Conteúdo de feed (rede social) — usa scroll infinito
  • Esconder o total — usuário precisa saber onde está

acessibilidade

06 · wcag aa
  • <nav aria-label="pagination">
  • aria-current="page" no número da página atual
  • Boundary buttons disabled (não só visualmente)
  • Foco visível em todos os controles

07 · html
