comportamentos

04
  • Botões − / + ajustam por step
  • Digitação direta sempre permitida
  • Setas ↑/↓ no input nativo funcionam
  • Botões boundary desabilitam quando min/max atingido
  • JS auto-init pelo components.js

quando usar

06

use

  • Quantidades (carrinho, estoque)
  • Configurações numéricas com range fixo
  • Quando step é pequeno (precisão por digitação)

não use

  • Valores visuais (volume, brilho) — usa slider
  • Range muito amplo (0-99999) — usa input + select de unidade
  • Valores não-numéricos

acessibilidade

07 · wcag aa
  • Botões com aria-label descritivo
