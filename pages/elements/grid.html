quando usar

04

use

  • Layouts macro — sempre encaixar em colunas do grid
  • Componentes ocupando spans inteiros (col 1-6, col 7-12)
  • Container max-width pra evitar linhas de texto longas demais

não use

  • Posicionamento absoluto pra alinhar — usa grid
  • Larguras fixas em px arbitrários — usa fr ou span

código

05
/* usar grid-12 com tokens */ .layout { display: grid; grid-template-columns: repeat(12, 1fr); gap: var(--spacing-05); max-width: var(--container-xl); margin: 0 auto; padding: 0 var(--spacing-06); }
