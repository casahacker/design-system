o bit

02 · unidade primitiva

BIT = 8px. Por que 8? É divisível por 2 e 4, encaixa em telas retina (×2, ×3), e cria um grid invisível que conecta padding, sizing, ícones e linha-altura.

8 BITs no horizontal · 32px cada · grid de 264px

modos de densidade

03

Componentes podem assumir 3 densidades. Para usar, troque os tokens base por seus equivalentes denser.

quando usar cada valor

04

use

  • Múltiplos do BIT — mesmo se for valor pequeno (8px sim, 7px não)
  • spacing-03 (8px) como gap default entre rows
  • spacing-09 (48px) como margin de section em desktop
  • spacing-05 (16px) como padding interno padrão

não use

  • Valores arbitrários (13px, 17px) — quebra o ritmo
  • spacing-01 / 02 em layouts macro
  • spacing-13 (160px) em mobile — quebra responsividade
  • spacings diferentes pra coisas iguais — gera inconsistência
