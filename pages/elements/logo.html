anatomia · 3 versões oficiais

01 · estrutura

A identidade tem três expressões. A escolha depende do espaço disponível e do contexto. Quando em dúvida, use a versão horizontal.

horizontal · primária · headers, papelaria, sites
vertical · empilhada · espaços quadrados, redes sociais
símbolo H · ícone · favicon, avatar, watermark, app icon

área de resguardo · clear space

02 · respiro

Mantenha pelo menos 1× a altura do H de espaço livre em volta do logotipo. Nenhum elemento (texto, imagem, borda) deve invadir essa zona.

Hachura verde = zona de resguardo · x = altura do H do logotipo

como aplicar

05 · do & don't

use

  • Sempre o SVG oficial · nunca recriar o logo manualmente
  • Maior contraste com o fundo: dark sobre claro, light sobre escuro
  • Respeitar a área de resguardo (1× altura do H)
  • Centralizar ou alinhar à grid de 8 px (BIT)
  • Para tamanhos < 24 px usar somente o símbolo H
  • Code (verde signature) só em comunicações institucionais

não use

  • Distorcer, rotacionar, inclinar ou alterar proporções
  • Aplicar sombras, gradientes, brilhos ou efeitos visuais
  • Recolorir fora das 5 variações oficiais
  • Sobrepor a fundos sem contraste suficiente (WCAG < 3:1)
  • Usar abaixo do tamanho mínimo (130/75/15 px)
  • Combinar com outros logos sem respeitar resguardo

favicon & app icon

06 · marca em miniatura

Em tamanhos miniatura (favicon, app icon, avatar) usa-se sempre o símbolo H isolado. Este site usa a versão pixelada do H (grid 3×3) como favicon, que mantém legibilidade até em 16 px.

código · uso em produtos

07

Os SVGs hard-codam fill em cores brand. Para usar com currentColor (auto light/dark), aplique-os como mask-image e pinte com background-color. É o padrão usado no header deste site.

<!-- 1. uso direto · cor fixa --> <img src="/assets/logos/casa-hacker/horizontal-dark.svg" alt="Casa Hacker" height="40"> <!-- 2. com currentColor · auto light/dark --> <span class="ch-logo" role="img" aria-label="Casa Hacker"></span> .ch-logo { display: inline-block; width: 140px; height: 18px; background: currentColor; -webkit-mask: url("/assets/logos/casa-hacker/horizontal-dark.svg") center/contain no-repeat; mask: url("/assets/logos/casa-hacker/horizontal-dark.svg") center/contain no-repeat; }
